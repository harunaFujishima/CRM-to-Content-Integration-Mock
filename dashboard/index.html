Approving a request calls POST /api/approvals/:id/approve on the CRM service. That service then calls POST /webhooks/crm-approval on the Content Platform service with an X-Webhook-Secret header, which writes a distribution record and returns it. This panel just polls both services' REST APIs to render what actually happened.
